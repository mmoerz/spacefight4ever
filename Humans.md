## 0. Guide to follow for agentic coding
- Rule 1: no ai generated memory files
- Rule 2: Short sessions. Each single feature should be implemented in a session.
- Rule 3: context engineering, agents.md ausbauen, sorgfaeltig aufbauen, dass sie den aktuellen kontext fuer die entwicklung wiederspiegeln
